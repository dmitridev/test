#Руководство по Api
POST /api/users/ возвращает список всех пользователей.
===
необходимые данные в теле запроса:

| имя |описание|  
|---|---|  
|token|сгенерированный токен, вкладка Токен в меню.|  
|key|сгенерированный ключ пользователя.|  
|type| json или xml|
